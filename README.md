# FUTURE_CS_01

## Vulnerability Assessment Report

### Target Website
OWASP Juice Shop (https://owasp-juice.shop)

### Assessment Type
Read-Only Vulnerability Assessment

### Scope
This assessment was conducted using passive and non-intrusive techniques. No exploitation, brute-force attacks, denial-of-service testing, or unauthorized access attempts were performed.

### Tools Used
- Nmap
- OWASP ZAP (Passive Scan)
- Browser Developer Tools

### Findings Summary

| ID | Finding | Risk |
|----|----------|------|
| F-01 | FTP Service Exposed | Medium |
| F-02 | Additional Web Service Exposed (Port 8080) | Low |
| F-03 | Strict-Transport-Security Header Not Set | Low |
| F-04 | Retrieved From Cache | Informational |
